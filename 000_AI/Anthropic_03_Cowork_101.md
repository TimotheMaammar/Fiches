# Introduction to Claude Cowork

Cowork est une évolution de Claude vers un mode agentique appliqué au "knowledge work" (analyse, rédaction, recherche, etc.).

Un peu comme pour Claude Code, le modèle peut agir et ne pas se cantonner au schéma question / réponse. C'est donc évidemment plutôt dédié aux actions concrètes (avec les mêmes possibilités que pour Claude Code niveau parallélisation et planification des tâches). 

Trois piliers : 
1) Plan : Claude propose un plan avant de commencer
2) Exécution : Claude exécute la tâche et peut travailler très longtemps
3) Connexion : Possibilité d'accès direct à d'autres outils

Il peut s'utiliser soit dans Claude Desktop soit avec une extension Chrome. 

La boucle est souvent la même : 
1) Décrire ce que l'on veut
2) Répondre aux questions de Claude
3) Laisser tourner 
4) Récupérer le résultat (le fichier généré par exemple
5) Possibilité de corriger en live 

L'environnement est isolé et cloisonné, avec validation pour les suppressions et un historique local. 

Trois modèles : 
- Opus => Très puissant mais coûteux
- Sonnet => Équilibré et choix par défaut
- Haiku => Rapide et léger
